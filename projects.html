FixFast Buddy - Vehicle Breakdown Assistance System

Second Year Group Project

May 2023 – May 2024

FixFast Buddy is a comprehensive breakdown assistance system designed to provide rapid and reliable help for vehicles experiencing issues on the road. Our system addresses the critical need for timely assistance, aiming to significantly reduce vehicle downtime and enhance driver safety.

Key features include real-time tracking of service providers, an intuitive user interface for quick requests, and an efficient dispatch system that routes assistance requests to the nearest provider. The integration of Google Maps API enhances the user experience by offering precise location tracking and updates.

Dehati - Decentralized Communication Android Application

Third Year Group Project

May 2024 – Ongoing

NBQSA 2024 Finalist - Tertiary Student Projects (Technology)

Dehati is an innovative Android application designed to facilitate secure, decentralized peer-to-peer messaging within local networks. By leveraging TCP/IP socket programming, Dehati allows users connected to the same Wi-Fi network to exchange messages directly, without the need for a central server or internet connection.

The application is built with a focus on privacy and efficiency, ensuring that communication remains secure and free from external interference. Key features include seamless local message exchange, a user-friendly interface for easy navigation, and a robust ping functionality that allows users to check the connectivity status between devices.


Kotlin Flutter Jetpack Compose TCP/IP Socket Programming
